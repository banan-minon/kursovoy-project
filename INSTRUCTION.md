# Руководство пользователя / Администратора

## Для покупателя:
1. Зайдите на главную страницу.
2. Воспользуйтесь фильтрами (Футболки, Худи) для выбора категории.
3. Нажмите на карточку товара для просмотра описания.
4. Нажмите "В корзину".
5. Перейдите в корзину (иконка 🛒), измените количество, если нужно.
6. Нажмите "Оформить заказ" (требуется авторизация).
7. Проверьте статус заказа в разделе "Мои заказы".

## Для администратора:
1. Авторизуйтесь под учетной записью с ролью `admin`.
2. Перейдите в раздел "Админ-панель" через шапку сайта.
3. Заполните форму (Название, Категория, Цена, URL картинки).
4. Нажмите "Опубликовать". Новый товар появится в общем каталоге.
5. Для удаления товара нажмите кнопку "Удалить" в списке товаров админ-панели.
